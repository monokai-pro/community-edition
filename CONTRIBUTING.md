# Monokai Pro

Monokai Pro themes are collections of color schemes, user interface colors and an icon pack consisting of custom-made icons. Monokai Pro themes are proprietary software, copyrighted and maintained by [Monokai](https://monokai.com).

## Contributing Community Editions

Monokai Pro Community Editions (CE) are themes maintained by the community. There are some rules for contributing a Community Edition, which you can find on [monokai.pro/contribute](https://monokai.pro/contribute).

## Instructions

When creating your own Community Edition, you can clone this GitHub repository and adjust the contents where needed. At a minimum, your Monokai Pro Community Edition repository should include the following files:

- [`README.md`](README.md): text for GitHub
- [`LICENSE.md`](LICENSE.md): Monokai Pro (CE) license
- [`INSTALL.md`](INSTALL.md): installation instructions for your theme
- [`screenshot.png`](screenshot.png): a high-res screenshot of your theme (minimum 2048px wide)
- [`icon.png`](icon.png): icon of the app you're creating your theme for (minimum 1024px wide, centered and padded)

Be sure to double-check your content, as some of it will be incorporated on the [Monokai Pro website](https://monokai.pro)

## Submit your Monokai Pro (CE) theme

When your theme is finished, please [submit an issue](https://github.com/monokai-pro/monokai-pro/issues/new) with a link to your repository. Your repository will then be added to the Monokai Pro organization with permissions to maintain it.
